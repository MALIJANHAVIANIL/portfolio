## ✨ Features
- **Responsive Design:** Fully responsive layout that looks great on mobile, tablet, and desktop devices.
- **Dark/Light Mode:** Seamless theme toggling for an optimal reading experience.
- **Modern Animations:** Smooth scrolling, fade-in animations, and interactive hover effects.
